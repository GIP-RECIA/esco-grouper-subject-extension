ESCO subject extension.

  • Adds the support for the notion of context. this context is defined by the values of an attribute.

  • Handles the mutlti valued attributes for the method Subject.getAttributeValue.
    This is usefull in order to be able to retrieve the mutlti-valued attributes via the ws.
    (see https://bugs.internet2.edu/jira/browse/GRP-257)
